Stage de première année



J'ai réalisé mon stage de première année chez Oasis Web Agency : société par actions simplifiées spécialisée dans le développement d’applications Web & Mobile, SEO, Stratégie Digitale, et UX/UI.

Réalisation d'un site internet en HTML, CSS et Javascript


Ma principale mission était de concevoir et développer un site web moderne, en utilisant les langages de base du web tels que HTML, CSS et JavaScript. J'ai travaillé avec l'aide de mon maître de stage et utilisé mes compétences acquises lors de cette année pour créer une expérience utilisateur convaincante. J'ai pu mettre en œuvre des fonctionnalités avancées en utilisant JavaScript, ce qui a permis d'améliorer l'interactivité et l'ergonomie du site.


Réalisation d'une application web en VueJs

Pendant mon stage, j'ai développé une application en Vue.js qui permet la prise de rendez-vous dans un salon de coiffure. Cette application
était connectée à une base de données Firebase, offrant ainsi une gestion centralisée et en temps réel des rendez-vous. Grâce à Vue.js, j'ai pu créer une interface utilisateur réactive et conviviale, simplifiant ainsi le processus de réservation pour les clients du salon. Cette expérience m'a permis de renforcer mes compétences en développement web et d'explorer les avantages offerts par Vue.js pour la création d'applications interactives et connectées à des bases de données.
